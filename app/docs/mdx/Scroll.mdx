
# Scroll Area

A native scroll container with stylable scrollbars. The required structure is `Root > Viewport > Content` — the `Scrollbar`s go **outside** the `Viewport`, as siblings within `Root`.

# Anatomy

```tsx
import { ScrollArea } from "gnome-ui/scroll-area";

<ScrollArea.Root>
  <ScrollArea.Viewport>
    <ScrollArea.Content />
  </ScrollArea.Viewport>
  <ScrollArea.Scrollbar orientation="vertical">
    <ScrollArea.Thumb />
  </ScrollArea.Scrollbar>
  <ScrollArea.Scrollbar orientation="horizontal">
    <ScrollArea.Thumb />
  </ScrollArea.Scrollbar>
  <ScrollArea.Corner />
</ScrollArea.Root>
```

# Examples

## Both Axes

When scrolling in both directions, add `<ScrollArea.Corner>` to cover the area where the two scrollbars intersect.

  </Tabs.List>

    <ScrollArea.ScrollAreaBothAxes />
  </Tabs.Panel>
  <Tabs.Panel className="p-0 outline-none" value="code">
    ```tsx
    export function ScrollAreaBothAxes() {
      return (
        <ScrollArea.Root className="h-52 w-96 rounded-xl border border-border overflow-hidden">
          <ScrollArea.Viewport className="h-full w-full">
            <ScrollArea.Content className="min-w-max">
              <table>{/* tabla ancha */}</table>
            </ScrollArea.Content>
          </ScrollArea.Viewport>
          <ScrollArea.Scrollbar orientation="vertical">
            <ScrollArea.Thumb />
          </ScrollArea.Scrollbar>
          <ScrollArea.Scrollbar orientation="horizontal">
            <ScrollArea.Thumb />
          </ScrollArea.Scrollbar>
          {/* Corner covers the intersection of the two scrollbars */}
          <ScrollArea.Corner />
        </ScrollArea.Root>
      );
    }
    ```
  </Tabs.Panel>
</Tabs.Root>

## Code Block

Ideal for code blocks with horizontal scroll. `min-w-max` on `Content` prevents text wrapping.

<Tabs.Root className="overflow-hidden rounded-xl border border-gray-200 bg-white shadow-sm" defaultValue="preview">

  </Tabs.List>
  <Tabs.Panel className="flex min-h-[240px] items-center justify-center p-6 outline-none" value="preview">
    <ScrollArea.ScrollAreaCodeBlock />
  </Tabs.Panel>
  <Tabs.Panel className="p-0 outline-none" value="code">
    ```tsx
    export function ScrollAreaCodeBlock() {
      return (
        <ScrollArea.Root className="w-96 rounded-xl border border-border bg-[#1e1e2e] overflow-hidden">
          <ScrollArea.Viewport className="h-full w-full max-h-52">
            {/* min-w-max prevents code wrapping */}
            <ScrollArea.Content className="min-w-max">
              <pre className="p-4 text-xs font-mono">
                {codeLines.map((line, i) => (
                  <div key={i} className="flex">
                    <span className="mr-4 w-6 text-right text-[#6c7086] select-none">
                      {i + 1}
                    </span>
                    <span className="text-[#cdd6f4]">{line}</span>
                  </div>
                ))}
              </pre>
            </ScrollArea.Content>
          </ScrollArea.Viewport>
          <ScrollArea.Scrollbar orientation="vertical">
            <ScrollArea.Thumb />
          </ScrollArea.Scrollbar>
          <ScrollArea.Scrollbar orientation="horizontal">
            <ScrollArea.Thumb />
          </ScrollArea.Scrollbar>
          <ScrollArea.Corner />
        </ScrollArea.Root>
      );
    }
    ```
  </Tabs.Panel>
</Tabs.Root>
## ScrollAreaContent

A container for the content of the scroll area.
Renders a `&lt;div&gt;` element.

Documentation: [Base UI Scroll Area](https://base-ui.com/react/components/scroll-area)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **className** | `string \| (...) => string)` | — | CSS class applied to the element, or a function that<br/>returns a class based on the component’s state. |
| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |

## ScrollAreaScrollbar

A vertical or horizontal scrollbar for the scroll area.
Renders a `&lt;div&gt;` element.

Documentation: [Base UI Scroll Area](https://base-ui.com/react/components/scroll-area)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **className** | `string \| (...) => string)` | — | CSS class applied to the element, or a function that<br/>returns a class based on the component’s state. |
| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |
| **orientation** | `"horizontal" \| "vertical"` | `'vertical'` | Whether the scrollbar controls vertical or horizontal scroll. |
