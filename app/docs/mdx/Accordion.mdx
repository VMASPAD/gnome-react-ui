## AccordionPanel

A collapsible panel with the accordion item contents.
Renders a `&lt;div&gt;` element.

Documentation: [Base UI Accordion](https://base-ui.com/react/components/accordion)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **className** | `string \| (...) => string)` | ‚Äî | CSS class applied to the element, or a function that<br/>returns a class based on the component‚Äôs state. |
| **render** | `ReactElement> \| ComponentRenderFn` | ‚Äî | Allows you to replace the component‚Äôs HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |
| **hiddenUntilFound** | `boolean` | `false` | Allows the browser‚Äôs built-in page search to find and expand the panel contents.<br/><br/>Overrides the `keepMounted` prop and uses `hidden="until-found"`<br/>to hide the element without removing it from the DOM. |
| **keepMounted** | `boolean` | `false` | Whether to keep the element in the DOM while the panel is closed.<br/>This prop is ignored when `hiddenUntilFound` is used. |

## AccordionRoot

Groups all parts of the accordion.
Renders a `&lt;div&gt;` element.

Documentation: [Base UI Accordion](https://base-ui.com/react/components/accordion)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **value** | `AccordionValue<Value>` | ‚Äî | The controlled value of the item(s) that should be expanded.<br/><br/>To render an uncontrolled accordion, use the `defaultValue` prop instead. |
| **defaultValue** | `AccordionValue<Value>` | ‚Äî | The uncontrolled value of the item(s) that should be initially expanded.<br/><br/>To render a controlled accordion, use the `value` prop instead. |
| **disabled** | `boolean` | `false` | Whether the component should ignore user interaction. |
| **hiddenUntilFound** | `boolean` | `false` | Allows the browser‚Äôs built-in page search to find and expand the panel contents.<br/><br/>Overrides the `keepMounted` prop and uses `hidden="until-found"`<br/>to hide the element without removing it from the DOM. |
| **keepMounted** | `boolean` | `false` | Whether to keep the element in the DOM while the panel is closed.<br/>This prop is ignored when `hiddenUntilFound` is used. |
| **loopFocus** | `boolean` | `true` | Whether to loop keyboard focus back to the first item<br/>when the end of the list is reached while using the arrow keys. |
