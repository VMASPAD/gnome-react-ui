# Switch

## SwitchRoot

Represents the switch itself.
Renders a `&lt;span&gt;` element and a hidden `&lt;input&gt;` beside.

Documentation: [Base UI Switch](https://base-ui.com/react/components/switch)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **className** | `string \| (...) => string)` | — | CSS class applied to the element, or a function that<br/>returns a class based on the component’s state. |
| **defaultChecked** | `boolean` | `false` | Whether the switch is initially active.<br/><br/>To render a controlled switch, use the `checked` prop instead. |
| **id** | `string` | — | The id of the switch element. |
| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |
| **value** | `string` | — | The value submitted with the form when the switch is on.<br/>By default, switch submits the "on" value, matching native checkbox behavior. |
| **disabled** | `boolean` | `false` | Whether the component should ignore user interaction. |
| **name** | `string` | — | Identifies the field when a form is submitted. |
| **nativeButton** | `boolean` | `false` | Whether the component renders a native `&lt;button&gt;` element when replacing it<br/>via the `render` prop.<br/>Set to `true` if the rendered element is a native button. |
| **required** | `boolean` | `false` | Whether the user must activate the switch before submitting a form. |
| **readOnly** | `boolean` | `false` | Whether the user should be unable to activate or deactivate the switch. |
| **inputRef** | `Ref<HTMLInputElement>` | — | A ref to access the hidden `&lt;input&gt;` element. |
| **checked** | `boolean` | — | Whether the switch is currently active.<br/><br/>To render an uncontrolled switch, use the `defaultChecked` prop instead. |
| **onCheckedChange** | `(...) => void` | — | Event handler called when the switch is activated or deactivated. |
| **uncheckedValue** | `string` | — | The value submitted with the form when the switch is off.<br/>By default, unchecked switches do not submit any value, matching native checkbox behavior. |

## SwitchThumb

The movable part of the switch that indicates whether the switch is on or off.
Renders a `&lt;span&gt;`.

Documentation: [Base UI Switch](https://base-ui.com/react/components/switch)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |

| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |
