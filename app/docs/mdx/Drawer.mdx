## DrawerRoot

Groups all parts of the drawer.
Doesn't render its own HTML element.

Documentation: [Base UI Drawer](https://base-ui.com/react/components/drawer)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **open** | `boolean` | — | Whether the drawer is currently open. |
| **defaultOpen** | `boolean` | `false` | Whether the drawer is initially open.<br/><br/>To render a controlled drawer, use the `open` prop instead. |
| **modal** | `boolean \| "trap-focus"` | `true` | Determines if the drawer enters a modal state when open.<br/>- `true`: user interaction is limited to just the drawer: focus is trapped, document page scroll is locked, and pointer interactions on outside elements are disabled.<br/>- `false`: user interaction with the rest of the document is allowed.<br/>- `'trap-focus'`: focus is trapped inside the drawer, but document page scroll is not locked and pointer interactions outside of it remain enabled. |
| **onOpenChange** | `(...) => void` | — | Event handler called when the drawer is opened or closed. |
| **onOpenChangeComplete** | `(...) => void` | — | Event handler called after any animations complete when the drawer is opened or closed. |
| **disablePointerDismissal** | `boolean` | `false` | Determines whether the drawer should close on outside clicks. |
| **actionsRef** | `RefObject<DrawerRootActions>` | — | A ref to imperative actions.<br/>- `unmount`: When specified, the drawer will not be unmounted when closed.<br/>Instead, the `unmount` function must be called to unmount the drawer manually.<br/>Useful when the drawer's animation is controlled by an external library.<br/>- `close`: Closes the drawer imperatively when called. |
| **handle** | `DialogHandle<Payload>` | — | A handle to associate the drawer with a trigger.<br/>If specified, allows detached triggers to control the drawer's open state.<br/>Can be created with the Drawer.createHandle() method. |
| **triggerId** | `string` | — | ID of the trigger that the drawer is associated with.<br/>This is useful in conjunction with the `open` prop to create a controlled drawer.<br/>There's no need to specify this prop when the drawer is uncontrolled (i.e. when the `open` prop is not set). |
| **defaultTriggerId** | `string` | — | ID of the trigger that the drawer is associated with.<br/>This is useful in conjunction with the `defaultOpen` prop to create an initially open drawer. |
| **children** | `ReactNode \| PayloadChildRenderFunction<Payload>` | — | The content of the drawer. |
| **swipeDirection** | `SwipeDirection` | `'down'` | The swipe direction used to dismiss the drawer. |
| **snapPoints** | `DrawerSnapPoint[]` | — | Snap points used to position the drawer.<br/>Use numbers between 0 and 1 to represent fractions of the viewport height,<br/>numbers greater than 1 as pixel values, or strings in `px`/`rem` units<br/>(for example, `'148px'` or `'30rem'`). |
| **snapToSequentialPoints** | `boolean` | `false` | Disables velocity-based snap skipping so drag distance determines the next snap point. |
| **snapPoint** | `DrawerSnapPoint` | — | The currently active snap point. Use with `onSnapPointChange` to control the snap point. |
| **defaultSnapPoint** | `DrawerSnapPoint` | — | The initial snap point value when uncontrolled. |
| **onSnapPointChange** | `(...) => void` | — | Callback fired when the snap point changes. |
