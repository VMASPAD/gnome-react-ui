## DrawerRoot

Groups all parts of the drawer.
Doesn't render its own HTML element.

Documentation: [Base UI Drawer](https://base-ui.com/react/components/drawer)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **open** | `boolean` | — | Whether the drawer is currently open. |
| **defaultOpen** | `boolean` | `false` | Whether the drawer is initially open.<br/><br/>To render a controlled drawer, use the `open` prop instead. |
| **modal** | `boolean \| "trap-focus"` | `true` | Determines if the drawer enters a modal state when open.<br/>- `true`: user interaction is limited to just the drawer: focus is trapped, document page scroll is locked, and pointer interactions on outside elements are disabled.<br/>- `false`: user interaction with the rest of the document is allowed.<br/>- `'trap-focus'`: focus is trapped inside the drawer, but document page scroll is not locked and pointer interactions outside of it remain enabled. |
| **onOpenChange** | `(...) => void` | — | Event handler called when the drawer is opened or closed. |
