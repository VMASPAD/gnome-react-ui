# Collapsible

## CollapsiblePanel

A panel with the collapsible contents.
Renders a `&lt;div&gt;` element.

Documentation: [Base UI Collapsible](https://base-ui.com/react/components/collapsible)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **className** | `string \| (...) => string)` | — | CSS class applied to the element, or a function that<br/>returns a class based on the component’s state. |
| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |
| **hiddenUntilFound** | `boolean` | `false` | Allows the browser’s built-in page search to find and expand the panel contents.<br/><br/>Overrides the `keepMounted` prop and uses `hidden="until-found"`<br/>to hide the element without removing it from the DOM. |
| **keepMounted** | `boolean` | `false` | Whether to keep the element in the DOM while the panel is hidden.<br/>This prop is ignored when `hiddenUntilFound` is used. |

## CollapsibleRoot

Groups all parts of the collapsible.
Renders a `&lt;div&gt;` element.

Documentation: [Base UI Collapsible](https://base-ui.com/react/components/collapsible)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **className** | `string \| (...) => string)` | — | CSS class applied to the element, or a function that<br/>returns a class based on the component’s state. |
| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |
| **open** | `boolean` | — | Whether the collapsible panel is currently open.<br/><br/>To render an uncontrolled collapsible, use the `defaultOpen` prop instead. |
| **disabled** | `boolean` | `false` | Whether the component should ignore user interaction. |
| **onOpenChange** | `(...) => void` | — | Event handler called when the panel is opened or closed. |
| **defaultOpen** | `boolean` | `false` | Whether the collapsible panel is initially open.<br/><br/>To render a controlled collapsible, use the `open` prop instead. |

## CollapsibleTrigger

A button that opens and closes the collapsible panel.
Renders a `&lt;button&gt;` element.

Documentation: [Base UI Collapsible](https://base-ui.com/react/components/collapsible)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **className** | `string \| (...) => string)` | — | CSS class applied to the element, or a function that<br/>returns a class based on the component’s state. |
| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |
| **nativeButton** | `boolean` | `true` | Whether the component renders a native `&lt;button&gt;` element when replacing it<br/>via the `render` prop.<br/>Set to `false` if the rendered element is not a button (e.g. `&lt;div&gt;`). |
