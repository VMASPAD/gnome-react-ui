# Toggle

A two-state button that can be either on or off. Useful for switching between two states or modes.

Documentation: [Base UI Toggle](https://base-ui.com/react/components/toggle)

import * as React from 'react';
import { Toggle } from '@base-ui/react/toggle';
import { Italic } from 'lucide-react';

const toggleBase =
  'inline-flex h-9 w-9 items-center justify-center rounded-lg border border-transparent text-sm font-medium text-foreground transition-colors hover:bg-accent hover:text-accent-foreground focus-visible:outline focus-visible:outline-2 focus-visible:outline-offset-2 focus-visible:outline-ring disabled:pointer-events-none disabled:opacity-50 data-[pressed]:bg-primary data-[pressed]:text-primary-foreground';

export function ToggleDefault() {
  const [pressed, setPressed] = React.useState(false);
  
  return (
    <Toggle 
      pressed={pressed} 
      onPressedChange={setPressed} 
      className={toggleBase}
      aria-label="Toggle italic"
    >
      <Icons.Italic className="size-4" />
    </Toggle>
  );
}

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **pressed** | `boolean` | — | Whether the toggle button is currently pressed.<br/>This is the controlled counterpart of `defaultPressed`. |
| **defaultPressed** | `boolean` | `false` | Whether the toggle button is currently pressed.<br/>This is the uncontrolled counterpart of `pressed`. |
| **disabled** | `boolean` | `false` | Whether the component should ignore user interaction. |
| **onPressedChange** | `(...) => void` | — | Callback fired when the pressed state is changed. |
| **value** | `string` | — | A unique string that identifies the toggle when used<br/>inside a toggle group. |
| **nativeButton** | `boolean` | `true` | Whether the component renders a native `&lt;button&gt;` element when replacing it<br/>via the `render` prop.<br/>Set to `false` if the rendered element is not a button (e.g. `&lt;div&gt;`). |
| **className** | `string \| (...) => string)` | — | CSS class applied to the element, or a function that<br/>returns a class based on the component’s state. |
| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |

## ToggleGroup

Provides a shared state to a series of toggle buttons.

Documentation: [Base UI Toggle Group](https://base-ui.com/react/components/toggle-group)

### Props

| Prop | Type | Default | Description |
| :--- | :--- | :--- | :--- |
| **value** | `readonly Value[]` | — | The open state of the toggle group represented by an array of<br/>the values of all pressed toggle buttons.<br/>This is the controlled counterpart of `defaultValue`. |
| **defaultValue** | `readonly Value[]` | — | The open state of the toggle group represented by an array of<br/>the values of all pressed toggle buttons.<br/>This is the uncontrolled counterpart of `value`. |
| **onValueChange** | `(...) => void` | — | Callback fired when the pressed states of the toggle group changes. |
| **disabled** | `boolean` | `false` | Whether the toggle group should ignore user interaction. |
| **orientation** | `Orientation` | `'horizontal'` | — |
| **loopFocus** | `boolean` | `true` | Whether to loop keyboard focus back to the first item<br/>when the end of the list is reached while using the arrow keys. |
| **multiple** | `boolean` | `false` | When `false` only one item in the group can be pressed. If any item in<br/>the group becomes pressed, the others will become unpressed.<br/>When `true` multiple items can be pressed. |
| **className** | `string \| (...) => string)` | — | CSS class applied to the element, or a function that<br/>returns a class based on the component’s state. |
| **render** | `ReactElement> \| ComponentRenderFn` | — | Allows you to replace the component’s HTML element<br/>with a different tag, or compose it with another component.<br/><br/>Accepts a `ReactElement` or a function that returns the element to render. |
